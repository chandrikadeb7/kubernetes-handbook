# Which Tool to Use for Orchestration



| **Team Size** | **No. of hosts/containers**  | **Tool to be used** |
| :--- | :--- | :--- |
| Fewer | Fewer | Nomad |
| Few | Few | Rancher |
| More | More | Docker Swarm |
| Bigger | Bigger | Mesos |
| Bigger | Bigger | Kubernetes |

## When to go serverless?

* Infrastructure from scratch
* Running on cloud
